- Cookie 与 Session 区别
<!-- 
1. Session是在服务端保存的一个数据结构, 用来跟踪用户的状态, 这个数据可以保存在集群、数据库、文件中.
2. Cookie是客户端保存用户信息的一种机制, 用来记录用户的一些信息, 也是实现Session的一种方式. 
3. 如果浏览器禁用Cookie, 同时session也会失效(可以通过其他方式实现, 比如url中传递session_id).
4. 维持一个回话核心就是客户端的唯一标识, 即session_id.
 -->
